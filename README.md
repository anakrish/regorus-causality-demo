# Regorus Causality Demo

Interactive browser demo and playground for the [Regorus](https://github.com/anakrish/regorus) causality feature.

**[Launch Demo](https://anakrish.github.io/regorus-causality-demo/)**

- [Scenario Tour](https://anakrish.github.io/regorus-causality-demo/index.html) — guided walkthrough of causal explanation scenarios
- [Playground](https://anakrish.github.io/regorus-causality-demo/playground.html) — custom policy, data, input, and entry point evaluation

Built from the [`causality`](https://github.com/anakrish/regorus/tree/causality) branch of [anakrish/regorus](https://github.com/anakrish/regorus/tree/causality) using WASM.
